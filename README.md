# Codeforces JavaScript (Node.js) Template

This repository provides a ready-to-use **Node.js template** for solving Codeforces problems.

It allows you to:
- Read input exactly like Codeforces (via stdin)
- Test solutions locally using `input.txt`
- Get output in `output.txt`
- Run using npm scripts
